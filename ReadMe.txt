Set up environment:

1. Use eclipse integrated with adt bundle.
2. Import project from dependancy folder and SpyCam project in workspace
3. Go to CameraService class and change server IP to desired server
4. Build project
5. Run project as android application
6. Install .apk file on desired device (emulator / smartphone)
7. Install PHP on server where you want to upload audio files
8. Keep SavePhoto.php file in www folder and create "uploads" folder in www
9. When application will start it will take photos on specific period and upload those images to server.
